# ImageAlchemy
This project aims to build a context-aware AI image editing system using diffusion models and transformer-based attention to perform realistic background replacement and character modification through a user-friendly interface.
Technologies Used
1. Core AI Frameworks
Python
PyTorch (deep learning framework)
2. Hugging Face Ecosystem
Hugging Face
Diffusers → for image generation/editing
Transformers → attention mechanisms
3. Model Used
Pretrained Stable Diffusion (inpainting pipeline)
Transformer-based attention layers (inside diffusion model)
4. Image Processing Libraries
OpenCV → image manipulation
PIL (Python Imaging Library) → image loading & saving
Matplotlib → visualization
5. Backend Development
Flask (for web application)
6. Frontend Technologies
HTML
CSS
JavaScript
7. Development Platforms
Google Colab → model training & experimentation
VS Code → application development
